## OIDC-Based Single Sign-On (SSO)

- Created an OIDC client using Authorization Code Flow.
- Configured redirect URIs and client settings.
- Tested SSO login and validated successful authentication.
- Enforced MFA during OIDC-based SSO login.

### Token Understanding
- ID Token: Used for user authentication.
- Access Token: Used for API authorization.
- Refresh Token: Used to maintain session continuity.

This lab demonstrates secure authentication and session handling using modern identity standards.
